# Random Colour Generator

A simple web application that generates random colors.

## Description

This project creates a random color generator. Click the button to generate a new random RGB color, which is displayed as text and applied as the background color of a box.

## Features

- Generates random RGB colors
- Displays the color code
- Visual representation of the color

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open `colourGen.html` in a web browser.
2. Click the "click me" button.
3. The color code will appear in the heading, and the box below will change to that color.

## Files

- `colourGen.html`: The main HTML file
- `colourGen.css`: Styles for the page
- `colourGen.js`: JavaScript for generating colors
